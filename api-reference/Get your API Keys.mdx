### 创建应用
如果你不是第一个使用DUIX API或者你想创建更多应用，点击网站右上角“头像” 图标，然后单击```Platform```去你的工作台查看。
![Platform.png](en-resource://database/511:1)
到工作台后，并在页面左上角点击 “齿轮” 图标。接下来，单击```API Keys```找到你已经创建的```app id```和```app key```。![my.png](en-resource://database/519:0)
单击```Creat APPKey```,设置```appkey名称```后，即可创建一个新的应用，点击复制时，去粘贴到文档中可查看您的```app id```和```app key```。![apikeys.png](en-resource://database/521:0)
注意：不要显示你的app-key给其他人。如果有人获得了您的访问权限app-key,他们可以像使用您的密码一样使用您的帐户。
