---
title: '创建您的第一个会话'
description: '开始使用DUIX API生成您的第一个对话'
---

### 创建账号
为了使用我们的API，首先你需要创建一个[DUIX.COM](https://duix.com/login)帐户，点击这里[去创建](https://duix.com/login)。
![login.png](https://github.com/zhouahua/duix.com/blob/main/images/login.png?raw=true)
### 登录工作台
登录成功后，点击 右上角“头像” 图标，然后再单击```Platform```去你的工作台。
![Platform.png](https://github.com/zhouahua/duix.com/blob/main/images/Platform.png?raw=true)
### 创建会话
然后在```Creat Session```页面创建你的第一个会话，你还可以选择自己喜欢的```Avatar``` 、```Background``` 、```Vioce``` ，可以选择想使用的```Models```及要创建的```Language```会话语言，若是选择非GPT4O的大模型，还可以设置```Openning remarks```,设置后在开始对话时数字人会先主动开始说话。
![create.png](https://github.com/zhouahua/duix.com/blob/main/images/create.png?raw=true)
### 获取appid/key、Session id
点击```Next```可获取```app id```和```app key```及```Session id```,点击”复制“图标复制成功后可取粘贴使用不要显示你的app key给其他人。如果有人获得了您的访问权限app key,他们可以像使用您的密码一样使用您的帐户。
![appid.png](https://github.com/zhouahua/duix.com/blob/main/images/appid.png?raw=true)
### 开始对接
最后，会话创建成功后，就可以去开始对接API进行运行查看效果啦，可以参考[API Docs](https://duix.com/apidoc)
![apidoc.png](https://github.com/zhouahua/duix.com/blob/main/images/apidoc.png?raw=true)
