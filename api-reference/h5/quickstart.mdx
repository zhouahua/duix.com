## 安装

```shell
npm i duix-guiji-light -S
```

## 快速开始

```js

import DUIX from 'duix-guiji-light'

const duix = new DUIX()

duix.on('intialSucccess', () => {
  // start session
  duix.start({
    conversationId: '',
    openAsr: true
  }).then(res => {
    console.info(res)
  })
})

// initialization
duix.init({
  sign: 'xxxx',
  containerLable: '.remote-container'
})

```
**注意：** 不要将duix实例放在Vue的data中，也不要放在react的state中（无需使其响应式）。
